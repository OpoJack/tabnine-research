# Hook Observations

Use this file to track whether hooks are useful, noisy, or counterproductive.

## What To Record

- which hook fired
- whether the message helped
- whether the hook was too noisy
- whether it should stay advisory or become stricter

## Questions To Ask

- Did the protected-branch warning fire at the right times?
- Did destructive-shell warnings catch anything useful?
- Did context hooks provide valuable reminders or just repeat the obvious?
- Are there hooks that should be merged or removed?
