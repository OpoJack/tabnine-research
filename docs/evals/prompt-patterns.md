# Prompt Patterns

Use this file to track prompt shapes that consistently work well with Tabnine CLI.

## Patterns To Compare

- command-only prompts
- command plus tagged file prompts
- explicit “read these files first” prompts
- URL comparison prompts using Web Fetch
- strict review prompts versus implementation prompts

## What To Record

- the prompt shape
- the task type
- what worked
- what caused drift or noise
- whether the result improved with narrower scope
